      MailTemplate Class

      Generates an email message for the specific (or matching) language from a localized Razor template file and the provided data object.

      Remarks

      The merge process makes these assumptions and conforms to the following conventions:

      • Every email template is identified by a string ID, such as EmailVerification, WelcomeMessage, etc.
      • Localized template files are named as templateId_languageCode.extension, such as Welcome_es-mx.html (separator characters can be customized).
      • Localized template files are formatted as valid HTML documents.
      • The contents of the <title> elements in the localized template files will be used as the email message subjects.
      • The contents of the <body> elements in the localized template files will be used as the email message bodies.
      • Template files can contain Razor syntax for data binding.
      • The specified language code must match the language code suffix in the file name.
      • If no template file with the language code suffix matching the specified language code is found for a particular email template ID, then an alternative language code will be used.
      • If a more specific language code (e.g. es-mx) is not implemented for the specified template, a template file with a more generic language code (e.g. es) will be tried.
      • A language map can be defined for more precise language code mapping.

      Example

      Zodiac_en-us.html file:
      XML
      <!DOCTYPE html>
<html lang="en">
<head>
<title>Welcome @Raw(Model.Zodiac)!</title>
<meta charset="utf-8">
</head>
<body>
<p>Hello @Raw(Model.Name),</p>
<p>
Your Zodiac sign is: @Raw(Model.Zodiac).
</p>
<p>
&copy; @Model.Year | <a href="#">Terms</a> | <a href="#">Privacy</a> | <a href="#">Unsubscribe</a>
</p>
</body>
</html>
      C# code:
      C#
      MailTemplate template = new();

Data data = new()
{
    Zodiac = "Leo",
    Name = "John",
    Year = 2025
};

// Load the en-US version of the Zodiac template from the Samples/Zodiac folder and merge it with data.
template.Load("Samples/Zodiac", "Zodiac", "en-US", ".html", data);

// Subject will hold the merged value of the title element.
string subject = template.Subject;

// Body will hold the merged value of the body element.
string body = template.Body;

// Language will hold the language code actually used by the template.
string language = template.Language;

      Constructors

      MailTemplate Initializes a new instance of the MailTemplate class.

      Properties

      Body Returns the text of the email HTML message body (after the data transformation performed by the Load(String, String, String, String, Object) call).
      Cached Indicates whether the pre-compiled template was loaded from the Razor engine's memory cache (after the data transformation performed by the Load(String, String, String, String, Object) call).
      Language Returns the real template language used for the specified template ID and language in a pretty format, such as xx-YY-ZZ (after the data transformation performed by the Load(String, String, String, String, Object) call).
      Subject Returns the email HTML message subject from the title element (after the data transformation performed by the Load(String, String, String, String, Object) call).
      Template Returns the original text of the localized HTML email template.

      Methods

      Load Loads a localized email template file for the specified (or matching) language and merges the template text with the specified data (if the data object is specified).
